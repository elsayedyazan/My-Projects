MISSION II

Rapport : Création d’une base de données

Accueil Mission I Mission III

Contexte:

L’objectif de cette mission est de commencer à exploiter la base de données de l’entreprise KDou pour pouvoir par la suite avoir des analyses et indicateurs pertinents.

Indicateur I:

Enoncé interprétation
Quelles sont les catégories de produit vendues par les fournisseurs Singapouriens ? Les catégories de produits vendus par les fournisseurs singapouriens nous donnent un indice précieux sur l’économie du pays, en mettant en évidence ses secteurs de renforcement, ainsi que ses capacités à proposer des produits de haute qualité ou plus de quantité ,donc moins coûteux pour l’achat.

Resultat SQL

Code SQL

SELECT DISTINCT Fournisseur.Pays, Produit.NomProd, Categorie.CodeCateg, Categorie.NomCateg FROM Fournisseur INNER JOIN (Categorie INNER JOIN Produit ON Categorie.CodeCateg = Produit.CodeCateg) ON Fournisseur.NoFour = Produit.NoFour WHERE (((Fournisseur.Pays)="Singapour"));

Indicateur II:

Enoncé interprétation
Quelle estle prix moyen de vente des produits, le prix minimum et le prix maximum. La comparaison des produits en utilisant trois critères (min, moyen, max) nous permet d’obtenir des informations sur les tendances des prix sur le marché en fonction de la demande des commandes. Elle aide également à contrôler la gestion des coûts, en ajustant les montants d'achat dans un domaine ou une catégorie de produits, soit en les augmentant, soit en les réduisan

Resultat SQL

Code SQL

SELECT Avg(DetailCommande.PrixUnitCli) AS MoyenneDePrixUnitCli, Min(DetailCommande.PrixUnitCli) AS MinDePrixUnitCli, Max(DetailCommande.PrixUnitCli) AS MaxDePrixUnitCli FROM DetailCommande;

Indicateur III:

Enoncé interprétation
Quels sont les produits conditionnés dans du carton ou dans du verre ? Le choix entre l'emballage en carton ou en verre peut avoir un impact significatif sur les coûts de transport et de stockage. Le carton, étant plus léger et plus facile à empiler, est souvent plus économique pour l'emballage et la logistique. En revanche, le verre, bien que plus lourd et fragile, offre une meilleure protection, notamment pour les produits liquides, en préservant leur qualité et en les protégeant des influences extérieures

Resultat SQL

Code SQL

SELECT Produit.NomProd, Produit.QteParUnite FROM Produit WHERE (((Produit.QteParUnite) Like "*carton*")) OR (((Produit.QteParUnite) Like "*verre*"));

Indicateur IV:

Enoncé interprétation
Dans quelles villes la clientèle est-elle la plus importante ? Un nouveau marché à forte demande offre une opportunité stratégique pour cibler précisément les consommateurs en élargissant ses offres ou en adaptant les produits aux besoins spécifiques du marché. Cela permet non seulement d’optimiser l’adéquation des produits avec les attentes des clients, mais aussi de mieux évaluer la rentabilité à long terme. Cela permet également de déterminer s’il est intéressant de poursuivre l’investissement dans ce marché

Resultat SQL

Code SQL

SELECT TOP 3 Client.VilleCli, Count("") AS NBClient FROM Client GROUP BY Client.VilleCli ORDER BY Count("") DESC;
