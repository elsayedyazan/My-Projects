MISSION I:

Rapport : Création d’une base de données

Accueil Mission II Mission III

Contexte

L'entreprise KDou propose à ses clients une large gamme de produits de restauration issus du monde entier. Elle sollicite notre équipe pour gérer sa base de données et exploiter ces informations afin d’orienter l’entreprise vers des pratiques plus respectueuses de l’environnement.

Bilan des données

La mission 1 consiste à prendre en main la base de données existante, gérée sur Excel.

Nous analysons cette base en recherchant diverses informations à l’aide d’un ensemble d’une douzaine de questions. Parmi ces questions, nous identifions des données telles que :


Le nombre de clients situés à Nantes,
Ou encore,le nombre de clients situés à Paris.

Erreurs dans la base

Grâce aux questions, nous nous familiarisons avec la base de données et constatons rapidement qu’elle comporte de nombreuses erreurs.

La plupart de ces erreurs proviennent de doublons. Cette mission met en évidence les limites d’Excel pour gérer une base de données de manière optimale.

En effet, une grande partie des erreurs aurait pu être évitée en utilisant un logiciel de gestion de base de données, qui empêche la création de clés primaires en double.

Perspectives d’amélioration

La meilleure amélioration envisageable serait de migrer vers un logiciel de gestion de base de données plus adapté, afin d’éliminer une grande partie des doublons. Cependant, certains doublons, comme "sao-paulo" et "são-paulo", proviennent de fautes d’orthographe ou de variations dans la saisie. Une vérification supplémentaire serait donc nécessaire pour identifier et corriger ces erreurs afin de supprimer efficacement les doublons.
